# 🃏 Cards Image Classification App

This project is a Streamlit-based deep learning app that classifies playing card images using a PyTorch ResNet18 model.

## 📂 Dataset
Automatically downloaded using kagglehub:
gpiosenka/cards-image-datasetclassification

## 🚀 Features
- Auto dataset download
- CNN (ResNet18 transfer learning)
- Image upload prediction
- Streamlit UI

## 🧠 Model
- PyTorch ResNet18
- Transfer Learning
- 53+ card classes

## ▶️ Run Locally

```bash
pip install -r requirements.txt
streamlit run streamlit_app.py
